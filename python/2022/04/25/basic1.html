2. 제어문 if 

if 조건1:
    조건 1이 참 일때의 실행문
elif 조건2:
    조건2가 참 일때의 실행문

    실행문

- 학점이 2보다 작으면 '학점경고'를 출력, 4이상이면 '우수', 나머지는 무시하시오.

grade = float(input("내 학점: "))

type(grade)

float
if grade >= 4:
    print('우수')
elif grade < 2:
    print('학점경고')
else:
    pass

- 점수가 90점 이상이면 'A', 80점 이상임면 'B', 70점 이상이면 'C', 60점 이상이면 'D', 나머지는 'F'로 출력하시오.

x = 70

if x >= 90:
    print('A')
elif x >= 80:
    print('B')
elif x >= 70:
    print('C')
elif x >= 60:
    print('D')
else:
    print('F')

- toeic 리스트에는 학생들의 토익점수가 들어있다. 점수가 950점 이상이면 '토익의신', 850점 이상이면 '좀하네', 700점 이상이면 '보통' 그 이하이면 '공부해라'를 출력하시오.

toeic = [250, 985, 850, 725, 600]

for i in toeic:
    print(i)

for i in toeic:
    if i >= 950:
        print(i, '토익의신')
    elif i >= 850:
        print(i, '좀하네..')
    elif i >= 700:
        print(i, '보통')
    else:
        print(i, '공부해라')
